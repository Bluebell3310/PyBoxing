# How to play

## Player 1
* Backword: <kbd>A</kbd>
* Forward: <kbd>D</kbd>
* Straight punch: <kbd>F</kbd>
* Uppercut: <kbd>G</kbd>
* Charge punch: <kbd>H</kbd>

* Forward: <kbd>→</kbd>
* Straight punch: <kbd>,</kbd>
* Uppercut: <kbd>.</kbd>
